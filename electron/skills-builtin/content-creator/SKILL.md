# 内容创作流程

## 1. 需求分析
- 确认内容类型（文章/文案/邮件/社媒）
- 确认目标受众
- 确认语气风格（正式/轻松/专业/幽默）
- 确认字数要求

## 2. 调研准备
- 用 `web_search` 搜索相关话题的最新信息
- 用 `recall` 检索用户之前的写作偏好
- 整理关键论点和数据

## 3. 大纲构建
先输出大纲结构，等用户确认后再正式创作。

大纲格式：
```
# 标题
## 引言（钩子）
## 核心观点 1
## 核心观点 2
## 核心观点 3
## 总结/行动号召
```

## 4. 逐段创作
- 每段有清晰的主题句
- 使用具体案例和数据支撑
- 段落间有自然过渡

## 5. 润色检查
- 检查逻辑连贯性
- 消除冗余表达
- 确保语气一致
- 用 `remember` 记住用户的写作偏好
