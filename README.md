# My Agent

一个人格化桌面 AI Agent — 有性格、有记忆、能成长的数字伙伴。

## 愿景

不只是一个 AI 工具，而是一个真正的数字伙伴：

- **人格化交互** — 有一致的性格特征和交流风格
- **持久记忆** — 记住你的偏好、项目上下文、历史对话，越用越懂你
- **主动协作** — 不仅被动回答，还能主动提醒、建议、推进任务
- **本地优先** — 数据存储在本地，隐私可控

## 技术栈

| 层级 | 技术 |
|------|------|
| 外壳 | Electron |
| 语言 | TypeScript 全栈 |
| 前端 | React + TailwindCSS |
| 存储 | SQLite + 向量数据库 |
| 核心 | Agent Loop（AsyncGenerator 事件流） |
