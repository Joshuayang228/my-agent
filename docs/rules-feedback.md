# 规则反馈

> 开发过程中 AI 发现规则不合理/缺失/冲突时记录到此文件。
> 累计 3 条以上反馈时，AI 会主动建议用户审视并修订规则。

## 反馈格式

```
### #序号 - 简述

- **规则文件**：哪个 .mdc / .md
- **具体条款**：哪条规则
- **问题类型**：不合理 / 缺失 / 冲突 / 过于严格 / 过于宽松
- **场景描述**：在什么情况下触发的
- **建议**：怎么改
- **状态**：待审视 / 已采纳 / 已驳回
```
