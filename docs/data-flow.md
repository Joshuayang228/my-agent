# 数据流转

> 数据/消息在各模块间的流转路径。链路复杂时及时更新此文档。

## 核心数据流

### 用户输入 → AI 响应（主链路）

```
用户输入文字
    │
    ▼
渲染进程（React）
    │ IPC: send user message
    ▼
主进程（Node.js）
    │
    ├─► 记忆系统：检索相关上下文
    │       ├─ 向量数据库：语义搜索历史
    │       ├─ SQLite：最近对话记录
    │       └─ 文件系统：项目记忆
    │
    ├─► 上下文组装
    │       ├─ System Prompt（固定指令）
    │       ├─ 长期记忆（检索结果）
    │       ├─ 当前会话（对话历史）
    │       └─ 临时上下文（当前文件等）
    │
    ├─► LLM 路由
    │       ├─ 选择 Provider
    │       └─ 发送请求
    │
    ▼
LLM API（流式返回）
    │
    ▼
Agent Loop（AsyncGenerator）
    │ yield AgentStreamEvent
    │
    ├─► 文本事件 ──► IPC 批量合并 ──► 渲染进程（流式显示）
    ├─► 工具调用事件 ──► 工具系统执行 ──► observe ──► 继续循环
    └─► 完成事件 ──► IPC 立即发送 ──► 渲染进程（结束标记）
```

### 工具调用链路

```
LLM 决定调用工具
    │
    ▼
Agent Loop：yield tool_call 事件
    │
    ▼
工具系统：查找工具定义
    │
    ├─ isReadOnly: true  → 直接执行
    ├─ isDestructive: true → IPC 弹窗请求用户确认
    │                          │
    │                     用户确认/拒绝
    │                          │
    │                     ├─ 确认 → 执行
    │                     └─ 拒绝 → 返回拒绝结果
    │
    ▼
工具执行结果
    │
    ▼
Agent Loop：yield tool_result 事件 → 继续 think
```

### IPC 事件分类

| 事件类型 | 发送策略 | 示例 |
|---------|---------|------|
| 文本 token | 批量合并（防淹没） | AI 输出的文字片段 |
| 工具开始 | 立即发送 | tool_call 开始 |
| 工具结束 | 立即发送 | tool_result 返回 |
| 完成 | 立即发送 | 本轮对话完成 |
| 错误 | 立即发送 | 异常错误 |
| 权限确认 | 序列化（一次一个） | 破坏性操作确认弹窗 |

## 数据持久化流

```
对话完成
    │
    ├─► SQLite：保存对话记录（消息 + 工具调用）
    ├─► 向量数据库：嵌入并索引对话内容
    └─► 文件系统：更新项目记忆（如有变更）
```
